## tachyons-ext

- .cream {#f5ead0};

```css

/* cream theme */

.cream { color: #f5ead0; }
.bg-cream { background-color: #f5ead0; }
.b--cream { border-color: #f5ead0; }

.hover-cream:hover,
.hover-cream:focus { color: #f5ead0; }

.hover-bg-cream:hover,
.hover-bg-cream:focus { background-color: #f5ead0; }
```
